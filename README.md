## Virtual Environment

1. Set up a virtual environment: `python -m venv .venv`.
2. Activate `.venv` and install requirements: `pip install --upgrade -r requirements.txt`.

## Usage

1. Active `.venv`.
1. Run `python bot.py`.

## Permissions

Requires permissions integer 68672:

- View Channels
- Send Messages
- Read Message History
- Add reactions
